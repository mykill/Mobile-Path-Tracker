A. Using the hip or the smartphone is placed in the pocket

First, the app will ask for the user’s gender and height in centimeters. The user can now start his/her first recording by clicking the “Start Record”. And in 5 seconds duration it should be putted in the pocket. The smartphone’s head/top should be at the bottom of the pocket and facing towards the user. After 5 seconds, the user may now start walking. After the user arrive in his/her destination, the user must click “Stop Recording” and the app will analyze the data gathered and will display the lists of path directions from where the user stopped to where the user started.

B. Using the hands while walking

First, the app will ask for the user’s gender and height in centimeters. The user can now start his/her first recording by clicking the “Start Record”. And in 5 seconds duration it should be in the hands. The smartphone’s head/top should be at the tip of the user’s hand and facing towards the user. After 5 seconds, the user may now start walking. After the user arrive in his/her destination, the user must click “Stop Recording” and the app will analyze the data gathered and will display the lists of path directions from where the user stopped to where the user started.
